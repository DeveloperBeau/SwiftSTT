# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc].

The full text of the covenant is hosted at the link above; please read it
before participating.

## Reporting

To report a concern about behaviour in this project's spaces (issues, pull
requests, discussions), [open an issue][issues] using the **Code of Conduct
report** template, or contact the maintainer through GitHub directly. The
maintainer treats reports as confidential.

[issues]: https://github.com/DeveloperBeau/SwiftSTT/issues/new/choose

## Enforcement

The maintainer reviews each report and applies the Enforcement Guidelines
described in section 4 of the covenant linked above. Possible outcomes range
from a private clarification to a permanent ban from project spaces.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
